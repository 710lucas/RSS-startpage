# RSS-startpage

A lightweight, simple, and highly customizable browser startpage focused on minimalism and productivity. It allows you to keep track of your favorite RSS feeds while providing a clean search interface and a beautiful aesthetic.

## 🚀 Features

* **Minimalist Dashboard:** A clean UI featuring a clock, a search bar, and your news feed.
* **RSS Integration:** Add and manage multiple RSS feeds to stay updated directly from your new tab.
* **Customizable Backgrounds:** Set your favorite wallpaper via URL or upload a local image (saved as base64).
* **Custom Search Engine:** Choose your preferred search provider (Google, DuckDuckGo, etc.).
* **No Backend Required:** Runs entirely in the browser. You can use it via GitHub Pages or simply open the index.html file locally.

## 🛠️ Installation & Usage

Since this is a client-side project, getting started is instant:

### Option 1: Local Usage
1. Clone the repository:
   `git clone https://github.com/710lucas/RSS-startpage.git`
2. Open `index.html` in any modern web browser.

### Option 2: GitHub Pages
You can host it for free using GitHub Pages:
1. Go to your repository Settings.
2. Navigate to Pages.
3. Select the main branch as the source and click Save.

## ⚙️ Customization

Click the gear icon in the top-right corner to open the Settings menu:
* **Search Engine:** Update the base URL (e.g., `https://duckduckgo.com/?q=`).
* **Background:** Paste a direct image link or upload a file from your computer.
* **RSS Feeds:** Add the XML/RSS links of the websites you follow.
